# Week 2 Module

## 📌 Overview
This repository contains the C source code for my Week 2 Module coursework. The primary focus of these programs is to build foundational data structure algorithms from scratch, emphasizing step-by-step logical execution and memory handling without relying on built-in sorting libraries. 

## 🧮 Problems Solved

### Searching Algorithms
**Find Student Roll Number (Linear Search):** A program to iterate through an unsorted list of registered students to find whether a particular student roll number exists.
**Search Product ID (Binary Search):** An algorithm designed to efficiently search for a target product ID within an online store's sorted inventory.

### Sorting Algorithms
**Arrange Exam Scores (Bubble Sort):** A sorting program to arrange students' exam scores in ascending order to prepare a result sheet.
**Arrange Library Book Numbers (Insertion Sort):** A program that takes book serial numbers one by one and places each new book in its correct sequential position.
**Rank Players by Score (Selection Sort):** An algorithm utilized by a gaming club to rank players based on their scores, sorting them from lowest to highest.

## 💻 Tech Stack
* **Language:** C 
* **Environment:** macOS / `gcc` compiler
* **Memory Management:** Utilized `malloc` and pointers for dynamic array allocation (e.g., in the Binary and Linear search implementations).
